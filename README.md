# Music Downloader

Music downloader is a simple bash script to download mp3 music from
the internet.

# Dependencies

Music Downloader depends on [Gum](https://github.com/charmbracelet/gum) from charmbracelet and [Yt-dlp](https://github.com/yt-dlp/yt-dlp)


# Running

```bash
chmod +x musicdl.sh
./musicdl.sh
```
